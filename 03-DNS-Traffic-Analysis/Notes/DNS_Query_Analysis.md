# DNS Query Analysis

## Transaction ID

Unique identifier used to match DNS queries with responses.

## Query Name

The hostname requested from the DNS server.

## Query Type

Defines the requested DNS record.

Common record types include:

- A
- AAAA
- CNAME
- MX
- TXT
